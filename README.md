# EXCEL-Sakles-DasbOARD

This repository contains an interactive Excel dashboard designed to visualize and analyze sales performance data from a Super Store. The dashboard provides a holistic view of key business metrics, enabling users to quickly identify trends, high-performing segments, and areas for improvement.
